## Quick Recommendation

| Need | Best Tool |
|------|-----------|
| **Complex scans with tables** | PaddleOCR (PP-Structure) |
| **Academic docs / equations** | olmOCR |
| **Simple forms** | docTR |
| **Basic printed text** | Tesseract |
